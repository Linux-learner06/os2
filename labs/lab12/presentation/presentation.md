# Цель работы

## Основная цель

Получить практические навыки настройки сетевых параметров и управления соединениями в ОС Linux.

# Ход выполнения работы

## Проверка конфигурации сети

- Команда **ip -s link** отобразила активные интерфейсы и статистику пакетов.  
- Интерфейс **enp0s3** имеет состояние **UP** и MAC-адрес `08:00:27:84:d7:ba`.  
- Проверка командой **ping 8.8.8.8** показала успешное подключение к Интернету.  

![Проверка конфигурации сети](Screenshot_1.png){ #fig:001 width=60% height=60% }

## Анализ маршрутов и адресов

- **ip route show** — основной шлюз **10.0.2.2**.  
- **ip addr show** — IPv4-адрес интерфейса **10.0.2.15/24**.  
- Дополнительно добавлен адрес **10.0.0.10/24** для **enp0s3**.  

![Отображение маршрутов и IP-адресов](Screenshot_2.png){ #fig:002 width=60% height=60% }

## Сравнение ip и ifconfig

- Утилита **ip** предоставляет больше данных, чем **ifconfig**.  
- Обе показывают состояние, MAC, IP, статистику пакетов.  

![Сравнение вывода ip и ifconfig](Screenshot_3.png){ #fig:003 width=60% height=60% }

## Проверка открытых портов

- Команда **ss -tul** показала активные порты:  

![Проверка открытых портов](Screenshot_3.png){ #fig:004 width=60% height=60% }

## Управление соединениями через nmcli

- Просмотр соединений: **nmcli connection show**  
- Созданы подключения:  
  - **dhcp** — автоматическое получение параметров  
  - **static** — ручная настройка IP: `10.0.0.10/24`, шлюз `10.0.0.1`  

![Создание и просмотр сетевых соединений](Screenshot_4.png){ #fig:005 width=60% height=60% }

## Переключение соединений

- Активация статического профиля: **nmcli connection up "static"**  
- Проверено изменение IP-адреса до **10.0.0.10**  
- Возврат к DHCP-подключению: **nmcli connection up "dhcp"**

![Переключение между соединениями](Screenshot_6.png){ #fig:006 width=60% height=60% }

## Изменение параметров через nmcli

- Отключено автоподключение профиля **static**.  
- Добавлены DNS-серверы: **10.0.0.10** и **8.8.8.8**.  
- Изменён IP: **10.0.0.20/24**, добавлен второй — **10.20.30.40/16**.  

![Изменение параметров соединения](Screenshot_7.png){ #fig:007 width=60% height=60% }

## Просмотр конфигурации через nmtui

- В **nmtui** отображены параметры профиля **static**:  
  - IP: **10.0.0.20/24** и **10.20.30.40/16**  
  - DNS: **10.0.0.10**, **8.8.8.8**  
  - Шлюз: **10.0.0.1**  

![Настройки профиля static в nmtui](Screenshot_8.png){ #fig:008 width=60% height=60% }

## Проверка в графическом интерфейсе

  - IP — **10.0.0.20**, **10.20.30.40**

  - Шлюз — **10.0.0.1**
