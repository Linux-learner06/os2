# Цель работы

## Основная цель

Получение навыков разметки дисков (MBR/GPT), создания файловых систем, настройки области подкачки и монтирования разделов в Linux.

# Процесс работы

## Подключение дисков и проверка

- Подключены диски **/dev/sdb** и **/dev/sdc**
- Проверка: `fdisk -l`
![Просмотр дисков](Screenshot_1.png){width=70%}

## Создание первичного раздела

- Создан раздел **/dev/sdb1**
- Размер: **300 MiB**
- Тип: Linux (MBR)
![Создание sdb1](Screenshot_3.png){width=70%}

## Создание расширенного и логического разделов

- Создан расширенный раздел **sdb2**
- Логический раздел **sdb5** на 300 MiB

![Расширенный и логический раздел](Screenshot_5.png){width=70%}

## Создание swap

- Создан **sdb6**
- Тип: **82 (Linux swap)**
- Активация: `mkswap`, `swapon`

![Создание swap](Screenshot_7.png){width=70%}

## Проверка структуры GPT-диска

- Используем диск **/dev/sdc**
- Разметка отсутствовала
- Создана новая GPT

![Проверка GPT](Screenshot_9.png){width=70%}

## Создание раздела GPT

- Раздел **sdc1**
- Размер: **300 MiB**
- Тип: **8300 — Linux filesystem**

![Создание GPT раздела](Screenshot_10.png){width=70%}

## Форматирование XFS

- Создана XFS на **sdb1**
- Метка: *xfsdisk*

![Форматирование XFS](Screenshot_12.png){width=70%}

## Форматирование EXT4

- Создана EXT4 на **sdb5**
- Метка: *ext4disk*
- Настроены параметры mount: `acl,user_xattr`

![Форматирование EXT4](Screenshot_13.png){width=70%}

## Ручное монтирование EXT4

- Создана точка `/mnt/tmp`
- Монтирование → проверка → размонтирование

![Ручное монтирование](Screenshot_14.png){width=70%}

## Автомонтирование через /etc/fstab

- Получены UUID через `blkid`
- Добавлена строка для **sdb1 → /mnt/data**
- Проверено через `mount -a`

![fstab настройка](Screenshot_15.png){width=70%}
